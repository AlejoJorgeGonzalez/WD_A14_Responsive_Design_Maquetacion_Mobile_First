# APLICANDO ESTILOS SECCION PLANES

Se crea los estilos de la sección de planes.

~~~css
/* Estilos del contenedor de planes */
.main-plans-container {
    width: 100%;
    min-width: 320px;
    padding-bottom: 70px;
    text-align: center;
}
~~~

Estilos en el titulo de la seccion de planes */

~~~css
.plans--title {
    width: 90%;
    min-width: 288px;
    height: auto;
    margin: 0 auto;
    margin-top: 50px;
}
~~~
